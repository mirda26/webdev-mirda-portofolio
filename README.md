# Proyek Portofolio Mirda Ramadani

Proyek ini merupakan tugas pembuatan portofolio sederhana menggunakan HTML, CSS, dan JavaScript. Isi proyek ini mencakup struktur halaman portofolio, tampilan visual, serta interaksi dasar seperti tombol dan elemen dinamis.

Repository ini berisi:
- index.html — struktur utama portofolio.
- style.css — pengaturan tampilan dan layout.
- script.js — fungsi interaktif sederhana.

Tujuan proyek ini adalah untuk mempraktikkan penggunaan GitHub, melakukan beberapa commit yang jelas, serta mempublikasikan proyek ke dalam repositori GitHub sesuai instruksi tugas.
